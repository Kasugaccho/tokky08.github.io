CSS

フォントサイズ

cssでフォントサイズを指定する場合、font-sizeと指定して、基本的にはpxかemの単位で表します。pxは固定値で大体15~18pxが標準です。emは今設定しているフォントサイズの何倍かを表す単位です。

文字の色

cssで文字の色を指定する場合、colorで指定できます。色名を英語で書くか、カラーコードを英数字で書くかで表せます。

文字寄せ

cssで文字寄せをする場合、text-alignで指定できます。left,center,rightが基本です。左詰か中央揃えか右詰のどれかです。

背景色

cssで背景色を指定する場合、background-colorで指定できます。color同様、色名かカラーコードで指定できます。

枠線

cssで枠線を指定する場合、borderで指定できます。borderでも色々種類がありますが、基本的なオーソドックスなものはsolidです。表を作成する時に使われたりします。

幅・高さ

cssで幅と高さを指定する場合、widthとheightと指定できます。基本的にpx指定です。また、インライン要素では指定できず、ブロック要素のみの指定となります。

画像の大きさ

cssで画像の大きさを指定する場合、ブロック要素でimgタグを囲んであげる必要があります。この時imgタグのwidthは100%ととし、ブロック要素のwidthを指定することで自由に変更させることができます。

余白

cssで余白を指定する場合、paddingとmarginで指定できます。borderが基準になっていてborderの内側がpaddingで外側がmarginで指定できるようになっています。指定方法は基本px指定です。上・右・下・左の順に指定できます。

配置-absolute,relative

cssで要素を自由に配置する場合、positionプロパティを用いてabsoluteまたは、relativeを使います。absoluteは基準を画面の左上とし、reraltiveは基準を本来配置される位置とします。親要素にreraltive、小要素にabsoluteを持ってくることで親要素の配置から操作できるようになります。top,bottom,left,rightで配置します。主に画像の上にテキストを持ってくる時に用います。

横並び

cssでブロック要素を横並びにする場合、flexを使います。floatでもできますが、flexの方が遥かに楽です。親要素にflex:display;をすることで小要素が横並びになります。そして、justify-content:space-between;で右端・左端と均等に配置してくれます。このセットでほとんどの配置が解決できると思います。
