HTML

HTMLの骨格

HTMLの骨組みの形です。headの内容は表示はされません。linkなどでcssを読み込む箇所です。bodyの中に表示させる内容を書いていきます。

h1~h6見出し

HTMLで見出しを表す時、h1~h6の要素があります。h1 > h2 > h3 > h4 > h5 > h6 の順に大きいです。実際使うのはh1,h2,h3あたりです。h6とかは使ったことがありません。

p 段落

HTMLで段落を表す時、pタグの要素があります。今実際にこの文章を打ってるタグもpタグです。普通の文章を打つ時にpタグを用います。改行する際は、brタグを後ろにつけるだけです。

ul,ol,li箇条書き

HTMLで箇条書きをする時、ul,ol,liタグを使います。liタグは実際に箇条書きする中身の要素で、ul,olはliタグを囲む要素です。ulはunorderと言って順番を決めない黒点で表します。olはorderなので順番を決めるため番号を振ります。

tableテーブル・表

HTMLで表を作成する時、tableタグを使います。tableは表の開始を表すタグで、この中にtr,th,tdタグを入れていきます。trタグは行を表し、この中にth,tdタグを入れていきます。thタグは見出しを表し。tdタグは内容を表します。

aリンク

HTMLでリンクをする時、aタグを使います。hrefでリンクを指定します。aタグはインライン要素です。インライン要素とは、幅と高さが決まっていません確か。なのでwidthなどを指定できなかったはずです。

img画像

HTMLで画像を挿入する時、imgタグを使います。srcに画像のリンクを指定します。altは画像の説明を入れます。これはW3Cで規定されていてなければエラーが吐かれます。ユーザーインターフェースのためにも画像が読み込めない時に対応して言語化する必要があります。

input

HTMLで書き込んだり入力する時、inputタグを使います。type属性でフォームの見た目が変わります。inputもまたインライン要素です。
